## 📖 Sobre o Jogo

**As Aventuras de Pérola** é um jogo de plataforma 2D desenvolvido para crianças, com gráficos fofos e mecânicas simples. Ajude a Pérola a navegar pelo fundo do mar, coletando estrelas brilhantes e evitando perigos como espinhos e águas-vivas.

### 🎯 Características

- ✨ **Personagem fofo** - Pérola é uma menina adorável com vestido colorido
- ⭐ **Sistema de pontuação** - Colete estrelas normais (+1) e douradas (+5)
- ❤️ **Sistema de vidas** - 3 vidas para completar o desafio
- 🌊 **Ambiente marinho** - Fundo do mar com bolhas, algas e nuvens
- 📱 **Controles adaptados** - Teclado e toque para dispositivos móveis
- 🎨 **Gráficos em canvas** - Todo desenhado com código, sem imagens externas

## 🎮 Como Jogar

### Controles

| Tecla | Ação |
|-------|------|
| ⬅️ ➡️ | Mover a Pérola para esquerda/direita |
| ⬆️ | Pular |
| **A / D** | Mover (alternativo) |
| **Espaço / W** | Pular (alternativo) |
| 📱 **Toque** | Pular (mobile) |
| 📱 **Arraste** | Mover (mobile) |

### Regras do Jogo

1. ⭐ **Colete estrelas** para acumular pontos
   - 🌟 Estrelas normais: +1 ponto
   - 🌟 Estrelas douradas: +5 pontos (mais raras)

2. 💀 **Desvie dos obstáculos** ou você perderá uma vida:
   - 🌊 Água-viva
   - 🌟 Espinho (estrela do mar vermelha)

3. ❤️ **Vidas** - Você começa com 3 vidas
   - Perde 1 vida ao colidir com obstáculos
   - Game Over quando todas as vidas se esgotam

4. 🏆 **Objetivo** - Faça a maior pontuação possível!
